# Project Overview

## Название
Arena AI (arena-hub)

## Назначение
Платформа из трёх AI-инструментов для аудита веб-сайтов, ориентированных на фаундеров, маркетологов и разработчиков.

## Инструменты
1. **RoastPage.ai** — AI-анализ лендингов: оценка копирайтинга, дизайна, CTA, конверсионного потенциала. Скоринг 0–100, 8 категорий.
2. **ShipChecklist.ai** — Pre-launch аудит: 50+ автоматизированных проверок (SEO, безопасность, производительность, юридические аспекты, доступность).
3. **PaywallAudit.ai** — Аудит страницы ценообразования: конверсионная психология, структура планов, стратегия монетизации. 10 категорий.

## Бизнес-модель
- Бесплатный уровень: ограниченный анализ (2–3 категории видны бесплатно).
- Полный отчёт: одноразовая покупка $7.99 за USDT (Polygon).
- Промо-коды для ограниченного бесплатного доступа.

## Целевая аудитория
Инди-хакеры, SaaS-фаундеры, маркетологи, разработчики.
